Librusik vs Librus Synergia
Pros (here Librusik is better than Librus Synergia):
Dark theme and pleasant user interface
Hide subjects with no grades
Independent average calculation - works even if school has disabled it
Attendances with detailed per-subject and per-semester frequency %% calculation
Countdowns to next holiday, next exam and school year end
You won't be logged out if you open Librusik occasionally
Has confetti
Cons (here Librusik sucks comparing to Librus Synergia):
Outgoing messages are unsupported
Some less used features are missing like point grades, homeworks, school notices etc.
Some implemented features may break under untypical conditions (Librus API is undocumented)
